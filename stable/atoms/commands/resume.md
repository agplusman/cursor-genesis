# Command: /resume

从 `NOTES.md` 恢复上下文，继续上次中断的任务。

## 触发场景

- 用户说"继续"、"接着做"、"恢复任务"
- 新对话中需要恢复上一轮长任务

## 执行步骤

1. 读取项目根目录的 `NOTES.md`
2. 提取当前任务、进度、关键文件和最后一条 checkpoint
3. 读取关键文件以重建上下文
4. 向用户确认："已恢复任务 [描述]，当前阶段 [X]，下一步 [Y]。是否继续？"
5. 用户确认后，从上次中断处继续执行

## 注意事项

- 如果 `NOTES.md` 不存在，告知用户没有可恢复的任务
- 恢复时优先加载关键文件，避免一次性加载全部上下文
- 如果任务已过期或环境变化大，建议用户重新评估而非盲目继续
